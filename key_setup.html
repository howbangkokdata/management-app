HOW Supabase Key Setup

หน้านี้ใช้ใส่ Supabase key ตอน runtime แทนการฝังคีย์ไว้ใน source code. ถ้าเลือก session-only คีย์จะหายเมื่อปิด tab/browser session.

หมายเหตุ: วิธีนี้ปลอดภัยกว่า hardcode ใน HTML แต่ service-role/secret key ก็ยังไม่ควรใช้ใน browser ระยะยาว. คำตอบถาวรคือ backend หรือ Supabase Edge Function ที่เก็บ secret ใน environment variable.
